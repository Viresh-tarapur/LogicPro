# Project Setup Instructions

## Backend
The Python backend is currently running on `http://localhost:8000`.
To restart it manually:
1.  Open a terminal in `v:/project/python extensions/extension/backend`.
2.  Run `python main.py`.

## Browser Extension
To load the extension:
1.  Open your browser (Chrome or Edge).
2.  Navigate to `chrome://extensions` (or `edge://extensions`).
3.  Enable **Developer mode** (toggle in the top right).
4.  Click **Load unpacked**.
5.  Select the directory: `v:/project/python extensions/extension`.
6.  The extension "AI Study Assistant" should appear.

## Usage
- Click the extension icon to open the popup.
- It will communicate with the local backend.
